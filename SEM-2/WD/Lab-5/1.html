Concept of WWW

  • It stands for World Wide Web
  • It is basically system of internet server that support specially formated documents
  • It is collection of text paages,digital photographs and animations we can access over the internet
  • User access the World Wide Web facilities via client called a web browser

HTTP Protocol

  1. HTTP stands for Hyper Text Transfer Protocol
  2. HTTP is based on the client-server architecture model and a ststeless request/response protocol
  3. 404 File Not Found is a common HTTP status code
